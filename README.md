# Fork CMS Geo
## Introduction
Geo names module

## Requirements
* Core: Fork CMS 3.9.4
* Module: Filter
* Module: Entities
* Module: Localization
* Library: GeoNames (add to your composer "nikkobautista/geonames" ~1.0)
